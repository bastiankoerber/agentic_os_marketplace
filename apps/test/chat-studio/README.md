# Chat Studio marketplace sample

This folder is a complete example of one marketplace app entry.

Expected structure:

- `meta.json` (required)
- `chat-studio-1.0.6.zip` (required, referenced by `meta.json`)
- `icon.svg` (optional marketing asset)
- `hero.svg` (optional marketing asset)

Repository structure for marketplace discovery:

- `apps/<category>/<app-slug>/...`

For this sample:

- category: `test`
- app slug: `chat-studio`
